Python基础知识

1、变量

  • 变量名只能包含字母、数字、下划线，不能以数字打头
  • 不要将Python关键字和函数名等Python保留用于特殊用途的单词用作变量名
  • 变量名应简短且具有描述性
  • 常用两种命名法，下划线命名法和驼峰命名法，如student_name, StudentName, studentName

print(message)

11、类

  • 面向对象编程中，编写各种类，基于这些类来创建对象，这些对象具备一些通用行为，即方法和属性
  • 封装、继承和多态是面向对象编程的三个主要特征
# 封装
class Dog():
    def __init__(self,name,age):
        """初始化属性name和age"""
        self.name = name
        self.age = age

    def sit(self):
        print(self.name.title()+" is now sitting.")

    def roll_over(self):
        print(self.name.title()+" rolled over!")

my_dog = Dog('willie',6)

print(my_dog.name)
print(my_dog.age)

willie
6

my_dog.sit()

Willie is now sitting.

my_dog.roll_over()

Willie rolled over!

# 继承
class SmallDog(Dog):
    def __init__(self,name,age):
        super().__init__(name,age)
    def eat(self):
        print(self.name.title()+" is now eatting.")

my_small_dog = SmallDog('andy',5)

my_small_dog.sit()

Andy is now sitting.

my_small_dog.eat()

Andy is now eatting.

# 多态 覆盖重写父类方法
# 继承
class BigDog(Dog):
    def __init__(self,name,age):
        super().__init__(name,age)
    def eat(self):
        print(self.name.title()+" is now eatting.")
    def sit(self):
        print('Big dog ' + self.name.title()+" is now sitting.")

my_big_dog = BigDog('alan',7)

my_big_dog.eat()

Alan is now eatting.

my_big_dog.sit()

Big dog Alan is now sitting.
